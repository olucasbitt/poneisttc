# Pôneis TTC Landing — Versão Refinada

Projeto em:
- React
- TypeScript
- Vite
- Tailwind CSS
- arquitetura por componentes

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Melhorias desta segunda versão
- visual mais premium
- cards mais refinados
- hero com métricas
- navegação mobile
- sombras e contrastes mais elegantes
- melhor hierarquia tipográfica
- blocos mais preparados para conteúdo real
